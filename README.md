# Nostra - E-commerce Website

A simple and responsive **E-commerce Website** built using **HTML**, **CSS**, and **JavaScript**. This project simulates an online shopping website with multiple pages, providing users with a clean and interactive browsing experience.

## Features
- 🏠 **Home Page** – Displays featured products, banners, and website highlights.
- 🛍️ **Products Page** – Showcases a collection of products with a clean and organized layout.
- 📞 **Contact Page** – Provides contact information and a simple way for users to get in touch.
- 📱 Responsive design for different screen sizes.
- 🎨 Modern and user-friendly interface.
- ⚡ Interactive features implemented using JavaScript.

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project was developed to practice front-end web development concepts, including webpage structure, responsive layouts, styling, and JavaScript-based interactivity. It demonstrates the fundamentals of building a multi-page e-commerce website using HTML, CSS, and JavaScript.
